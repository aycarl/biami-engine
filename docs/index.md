# Biami Engine Documentation

Welcome to the Biami Engine documentation! This directory contains essential resources for understanding the architecture, decisions, and future roadmap of the Biami mobile survey platform.

The roadmap details the 13-week transition from a functional core application to a production-grade, autonomous cloud architecture. It covers the evolution of infrastructure, serverless integration, security, and observability.

ADRs capture the *why* behind key technical decisions in the `biami-engine` repository. Review this section to understand the rationale for our framework choices, deployment models, and other architectural decisions.

### [AI Agent Instructions](../Agents.md)
Guidelines and architectural context specifically designed for AI coding agents and assistants working in this repository. It covers our tech stack, coding standards, and dynamic data handling practices.

### [Main README](../README.md)
The project's main README file, containing high-level information, setup instructions, and the general purpose of the Biami Engine.
